# Turn Based Game (Godot)

Simple turn based combat game using godot engine

Shows the following gameplay systems:

- Main Menu system
- Turn based combat loop
- Player attack and heal ability
- Enemy Attack Behavior
- Health bar with updates after each turn
- Victory and Defeat conditions

Created using:
- Godot
- GDScript

Future improvements:
- Difficulty setting from menu
- Critical hits
- Multiple enemy types
- Sound effects
- Character design
- Improved UI
- Graphic effects
